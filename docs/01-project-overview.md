# Project Overview

## Project Name
**Generate** - A Laravel 12 application for company data management and form building

## Technology Stack
- **Backend**: Laravel 12
- **Frontend**: Blade Templates + Tailwind CSS 4.0
- **Database**: SQLite (development) / MySQL/PostgreSQL (production)
- **Asset Bundling**: Vite 6.0
- **QR Code Generation**: TBD (SimpleSoftwareIO/simple-qrcode or similar)

## Project Goals

### Primary Features
1. **Company Data Management**
   - Store and manage company information
   - Brand identity management (colors, logo)
   - Social media links
   - QR code generation for company profile pages

2. **Form Builder**
   - Create custom forms and surveys
   - Full customization options for admins
   - Multiple question types
   - Response collection and management
   - Analytics and reporting

## Design Principles
- **Simple**: Clean, intuitive user interface
- **Modern**: Using latest Laravel features and modern frontend practices
- **Scalable**: Architecture that can grow with requirements
- **User-Friendly**: Easy to use for both admins and end users

## User Roles
- **Admin**: Full access to all features
- **User**: Can fill forms and view company profiles

## Development Phases

### Phase 1: Company Data Management
- Database schema for company data
- Admin interface for managing company information
- Public company profile page
- QR code generation and display

### Phase 2: Form Builder
- Database schema for forms and responses
- Form builder interface
- Form rendering and submission
- Response management and analytics
